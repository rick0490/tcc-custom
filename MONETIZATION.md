# Tournament Control Center - Monetization Strategy

**Comprehensive monetization roadmap for the TCC-Custom platform.**

Last Updated: December 2024

---

## Executive Summary

Tournament Control Center is a full-stack tournament management system with unique differentiators: physical Stream Deck integration, multi-display support (TV overlays, brackets, flyers), real-time updates, and a mobile-first participant experience. This document consolidates all monetization strategies from SaaS subscriptions to sponsor revenue streams.

**Revenue Potential:** $108,600 - $207,996/year (combined sponsor + SaaS revenue)

## 3. Sponsor Revenue System

### Current Capabilities

The sponsor overlay system supports:
- 6 display positions (4 corners + 2 banners)
- Configurable size (50-500%), opacity, border radius
- Sequential or random rotation
- Timer-based show/hide cycling
- WebSocket real-time updates
- Match and Bracket display targeting

### Position Pricing

| Position | Tier | Suggested Price/Event |
|----------|------|----------------------|
| top-left | Premium | $400-600 |
| top-right | Premium | $400-600 |
| bottom-left | Standard | $200-300 |
| bottom-right | Standard | $200-300 |
| top-banner | Budget | $100-150 |
| bottom-banner | Budget | $100-150 |

### Premium Sponsor Features

**Featured Sponsor Mode** (3x standard rate)
- 200%+ logo size
- Exclusive full-screen takeover during breaks
- Logo on both match AND bracket displays
- Priority position in rotation

**Grand Finals Spotlight** ($500-1000/event)
- Animated logo effect (pulse, glow)
- "Grand Finals brought to you by [Sponsor]" ticker
- Exclusive visibility during finals
- Winner announcement includes sponsor branding

**Winner Screen Branding** ($200-400/event)
- Large sponsor logo on winner announcement
- "Congratulations! Powered by [Sponsor]" text
- Screenshot-worthy moment for social sharing

### Time-Based Packages

| Time Slot | Duration | Price |
|-----------|----------|-------|
| Early Tournament | First hour | $150-200 |
| Mid Tournament | Hours 2-4 | $300-400 |
| Finals Hour | Last hour | $500-600 |
| Full Event | All slots | $800-1000 |

### Recurring Sponsor Subscriptions

| Frequency | Events/Month | Price | Per-Event |
|-----------|--------------|-------|-----------|
| Single Event | 1 | $300 | $300 |
| Monthly (4x) | 4 | $1000 | $250 |
| Quarterly (12x) | 12 | $2500 | $208 |
| Annual (48x) | 48 | $8000 | $167 |

### Engagement Features

**QR Code Overlay** (+$100-200)
- QR code linking to sponsor URL/discount
- Scan tracking for ROI metrics
- "Scan for exclusive discount" CTA

**Sponsor Ticker Integration**
- Auto-inject sponsor messages in ticker rotation
- "Thanks to [Sponsor] for supporting tonight's event!"
- Scheduled ticker messages at key moments

**Giveaway/Prize Sponsor**
- "[Sponsor] Prize Pool Contribution" recognition
- Logo on prize distribution display
- Recognition in tournament narrative/recap

## 11. Competitive Positioning

### vs. Start.gg
- Lower price point for smaller events
- Physical display integration (unique)
- Stream Deck support (unique)
- Simpler interface for locals
- Focus on bar/venue tournaments

### vs. Challonge
- Professional display system
- Real-time TV overlays
- Hardware controller support
- Venue-specific features

### vs. Smash.gg
- More accessible pricing
- Not limited to fighting games
- Better for bar/venue environments
- Hardware integration

**Key Differentiators:**
1. Physical Stream Deck integration
2. Multi-display TV system
3. Offline-capable local database
4. Venue-focused feature set

## 12. Success Metrics

### North Star Metrics
- Monthly Active Tournaments (MAT)
- Average Revenue Per User (ARPU)
- Net Revenue Retention (NRR)

### Leading Indicators
- Free to paid conversion rate (target: 5-10%)
- Trial to paid conversion (target: 25%)
- Monthly churn rate (target: <5%)
- Sponsor renewal rate (target: 60%+)

### Quick Wins to Implement First
1. Add "Powered by TCC" watermark to free tier
2. Implement participant limits
3. Create upgrade prompts when limits reached
4. Add Stripe integration
5. Build pricing page

| Risk | Impact | Mitigation |
