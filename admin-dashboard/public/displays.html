Setup Instructions

1
Configure Display Device

On your Raspberry Pi or display device, run the display client script to register with this admin panel.

2
Assign Display View

Once registered, select which MagicMirror view (Match, Bracket, or Flyer) should be shown on each display.

3
Monitor Status

Displays send heartbeats every 30 seconds. Green indicates online, red indicates offline or disconnected.
